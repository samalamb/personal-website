Bloccit

Goal

Bloccit was a project that I took on when learning the programming language Ruby in the Rails Web Development course at Bloc.

Execution

Bloccit uses the programming language Ruby along with the Web Application Framework Rails. This uses a dynamic database that the end user can add to. At it's core Bloccit is a Reddit like clone that can be accessed by anyone, but actual users have the ability to post their own content, comment on whatever they'd like and have their activity stored on their profile this being achieved with the database created using Rails and SQL.

Conclusion

The end product is a very nice website that anyone can go and use. They will be able to post their ideas and cool things that they find online. It was really beneficial to see how to make a database; how to relate keys and items and whatnot.
