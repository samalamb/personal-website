Alexa Project

Goal

During my time at Bloc I took on the Alexa Project. The main goal was to make a Skill that works on the Alexa Skill Platform.

Execution

Alexa runs on off Amazons Web Services. The basics would run off of Amazons Lambda service where you can store your main code either inline or as a zip file. You have to properly set the Lambda up so that the main Alexa Skill will be able to access the code.

Conclusion

At the end of the project I was able to launch my skill successfully. Nothing amazing, but if you like the show 30 Rock the skill will spit some trivia at you everytime you ask it to.
